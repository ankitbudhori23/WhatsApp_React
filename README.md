# WhatsApp_React_App

## WhatsApp clone with React and Firebase
## Run Locally

Clone the project

```bash
  git clone https://github.com/ankitbudhori23/WhatsApp_React.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
